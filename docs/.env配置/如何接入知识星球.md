# 如何接入知识星球

## 📋 准备工作

1. **微信公众号账号**: 需要有已加入的知识星球（[知识星球](https://wx.zsxq.com/)）

---

## 🔧 详细步骤

### 步骤 1: 登录知识星球

1. 打开浏览器,访问 [https://wx.zsxq.com/](https://wx.zsxq.com/)

### 步骤 2: 打开开发者工具

登录成功后,按下面的快捷键打开开发者工具:

- **Windows/Linux**: `F12` 或 `Ctrl + Shift + I`
- **macOS**: `Command + Option + I`

或者右键点击页面 → 选择"检查"(Inspect)

![](https://cdn.ziliu.online/images/2025/11/28a25169-f78e-42ee-a771-1dbe5f377a7a.jpg)

### 步骤 3: 切换到 Network 标签

在开发者工具中,点击顶部的 **"Network"**(网络)标签

![](https://cdn.ziliu.online/images/2025/11/5dced02f-c424-4e51-8cde-7c104494da3f.jpg)

### 步骤 4: 刷新页面并找到请求

1. 确保 Network 标签已打开
2. **刷新页面** (按 `F5` 或 `Ctrl+R` / `Command+R`)
3. 等待页面加载完成,你会看到很多网络请求
   
### 步骤 5: 过滤并查找主请求

在 Network 标签的**过滤框**中输入: `hashtags`

随便找一个请求，并点击

![](https://cdn.ziliu.online/images/2025/11/b985efc9-f064-4ccc-aa66-68d7d260795c.jpg)

### 步骤 6: 查看请求头信息

点击某个请求后,右侧会显示详细信息:

1. 点击 **"Headers"**(请求头)标签
2. 向下滚动找到 **"Request Headers"**(请求头)部分

### 步骤 7: 复制 Cookie

在 Request Headers 中找到 **`Cookie:`** 字段:

1. 点击 `Cookie:` 后面的值
2. **右键 → "Copy value"**(复制值)
   - 或者直接选中整行文本并复制

**重要**: Cookie 很长,确保复制完整!

![](https://cdn.ziliu.online/images/2025/11/3812dd5b-3637-47d0-903a-9238047271af.jpg)

---

## 💾 保存到配置文件

### 添加配置

在 `.env` 文件中配置:

```bash
ZSXQ_COOKIE=你复制的Cookie值
```
---

## ⚠️ 注意事项

### 1. Cookie的有效期

- Cookie 通常有效期为 **7天左右**
- 过期后需要重新获取
- 程序会自动检测 token 是否过期并提示
